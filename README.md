# wechat_crawler
## wechat_func.py
1. to_html()可將txt檔轉成html檔，方便後續解析
2. wechat_catch()可將上述得到的html檔解析成以下資料
    </br>◆ 文章網址(url)
    </br>◆ 文章標題(title)
    </br>◆ 文章發布日期(date)
    </br>◆ 文章作者(author) ※目前僅顯示原創文章
    </br>◆ 文章全文(content)
    </br>◆ 文章多媒體數量(media)
    </br>◆ 文章發布帳號(account)
3. 最終檔案將存成csv檔
